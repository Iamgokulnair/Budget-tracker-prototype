# IC01 Budget Tracker - Working Prototype

**The Problem:** Managing 15+ team members across 5 TLs with dual budget pools (Team + Connectivity) in Excel. Monthly reconciliation takes hours and leadership visibility is point-in-time, not real-time.

**The Solution:** Real-time budget dashboard with TL-level analytics, visual trend spotting, Excel import, and attrition handling.

**Try It Live:** [Coming soon - setting up GitHub Pages]

## Key Features
- ✅ Dual budget tracking (Team + Connectivity)
- ✅ TL-level spend analysis with filtering
- ✅ Monthly trend visualization (Spend Theme)
- ✅ One-click Excel import (hidden upload feature)
- ✅ Automated attrition adjustments
- ✅ Export for leadership reviews

## Quick Start
1. Open the live demo link above
2. Click "Configuration" to set budget per head
3. Add team members or use the hidden ⬆ button to import Excel
4. Track expenses and view "Spend Theme" for visual analytics

## Tech Stack
- Pure HTML5, CSS3, JavaScript (ES6+)
- Chart.js for visualizations
- SheetJS for Excel import
- localStorage for data persistence

## Status
Working prototype, actively used by IC01 team. Built to validate concept and gather feedback for potential VGM-wide scaling.
